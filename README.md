# power-bi
📊 Car Sales Analysis | Power BI Project

This project focuses on analyzing a car sales dataset to extract meaningful insights about customer behavior, sales performance, and pricing patterns.

🔍 Project Overview

Cleaned and transformed raw data for analysis

Built a structured data model with relationships

Created calculated columns & DAX measures

Designed an interactive dashboard for business insights

📈 Key Analysis

Customer segmentation based on income & demographics

Sales performance by brand, model, and region

Price distribution and its impact on purchasing decisions

Trends and patterns in car sales

🛠 Tools & Skills

Power BI

Data Modeling

DAX

Data Cleaning & Transformation

💡 Objective

To demonstrate how raw data can be transformed into actionable insights that support business decision-making.
